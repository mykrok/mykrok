# Tutorials

Step-by-step guides to learn MyKrok.

These tutorials will walk you through the basics of backing up your fitness activities and exploring your data.

## Available Tutorials

### [Getting Started](getting-started.md)

Your first steps with MyKrok:

- Installing MyKrok
- Setting up Strava API access
- Authenticating and syncing your first activities
- Viewing your data

### [Exploring Your Data](exploring-your-data.md)

Learn to use the interactive map browser:

- Navigating the map view
- Filtering activities by date and type
- Viewing activity details and tracks
- Using the statistics dashboard
